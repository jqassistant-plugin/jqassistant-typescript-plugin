# `:TS:Value:Function` Node

-> represents a function value

**Examples:**
```ts
let x = () => { 
  // ...
};
let y = function() { 
  // ...
};
```

## Properties

| Name                      | Description                                      |
| ------------------------- | ------------------------------------------------ |
| `arrowFunction` (boolean) | indicates, if the function uses the arrow syntax |
